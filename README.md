#Airbnb de Estudos
Problema
  Pessoas querem estudar juntas, mas não sabem com quem.

Solução
  Match de pessoas por:
    disciplina
    nível
    horário
    estilo (silencioso vs colaborativo)
    
Recursos
  PerfilEstudo
  SessaoEstudo
  Match

REST
  GET /perfis
  POST /perfis
  PUT /perfis/{id}
  DELETE /perfis/{id}
  GET /matches
  GET /sessoes
  POST /sessoes
  PUT /sessoes/{id}
  DELETE /sessoes/{id}
